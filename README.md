# Text-Editing-AS-Imitation-Game
This repository is for the paper Text Editing as Imitation Game
## Methods
+ seq2seq_lstm
+ lstm
+ lm_lstm
+ lm

## Language Models
+ bert-base-uncased
+ bert-large-uncased
+ albert-base-v2
+ albert-large-v2
+ roberta-base
+ roberta-large
+ xlm-roberta-base
+ xlm-roberta-large
+ funnel-transformer/large
+ funnel-transformer/xlarge

## Directory
+ **main/env** - Game Environment
+ **main/res** - Resource
+ **main/src** - Source Code
```
Text-Editing-AS-Imitation-Game/
├── README.md
└── main
```

## Dependencies
+ python >= 3.8.5

## Setup
Please ensure required packages are already installed. A virtual environment is recommended.
```
$ cd Text-Editing-AS-Imitation-Game
$ cd main
$ pip install pip --upgrade
```

## Run
Before training, please take a look at the **config.py** to ensure training configurations.
```
$ cd main
$ vim config.py
$ python main.py
```
